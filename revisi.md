Design a modern SaaS-style photography marketplace platform called Lensora.

The platform helps users discover and book photographers based on style, budget, location, and availability.

The design should closely follow modern SaaS landing page aesthetics.

Visual style:

- Clean
- Premium
- Modern
- Professional
- Friendly
- Trustworthy
- Conversion focused

The overall layout should resemble a modern startup website rather than a photography portfolio website.

Use:

Primary Color:
Blue (#2563EB)

Secondary Color:
Yellow (#FACC15)

Background:
White (#FFFFFF)

Section Background:
Light Gray (#F8FAFC)

Text:
Dark Gray (#111827)

The design should include:

- Rounded corners
- Soft shadows
- Large spacing
- Modular sections
- Clean cards
- Minimal icons
- Professional typography
- Modern UI components

Avoid:

- Magazine style layouts
- Asymmetrical creative layouts
- Dark themes
- Luxury fashion aesthetics
- Overly artistic compositions

The experience should feel like:
Airbnb + Stripe + Modern SaaS Marketplace
